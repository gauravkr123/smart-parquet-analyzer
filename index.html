Cannot open directly as a file

DuckDB-WASM needs an HTTP server to load its WebAssembly engine.
Browsers block this when opening HTML files directly (file://).

Quick fix — run one of these in this folder:

python3 -m http.server 8080
npx serve .

Then open http://localhost:8080

Or just use make serve if you have Make installed.
